# Stitch Prompt: Admin Login Page

**Intent:** Build a professional and secure admin login page for "Score Game," a video game store.

**Vibe & Theme:**
- **Mood:** Professional, secure, and clean with a subtle "gaming" aesthetic.
- **Theme:** Dark mode background (`#0f172a` or similar) with high-contrast accents (e.g., neon blue or purple).
- **Typography:** Sans-serif, clean, and highly readable.

**Page Structure & Elements:**
1. **Centered Login Card:**
   - A modern, slightly translucent card with a subtle border or glow effect.
   - **Header:** "Score Game Admin Dashboard" in a bold, prominent font.
   - **Sub-header:** "Restricted Access - Store Employees Only" in a smaller, secondary text style.
2. **Information Section:**
   - A brief, clear note: "Only pre-approved Google accounts can access this area. New accounts must be manually approved in the database. Contact the store owner if you need access."
3. **Primary Action:**
   - A large, high-contrast "Sign in with Google" button. Use the official Google logo if possible, or a clear "G" icon.
   - Button style: Rounded corners, hover effects, and clear focus states.
4. **Navigation:**
   - A "Return to Main Website" link at the bottom of the card or as a separate footer element.
5. **Responsiveness:**
   - Ensure the layout is fully responsive, looking centered and polished on mobile, tablet, and desktop.

**Style Keywords:** Minimalist, high-performance, mobile-friendly, secure, gaming-vibe.
