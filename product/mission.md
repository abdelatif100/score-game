# Product Mission

## Pitch
Score Game is a video game store information website that helps store customers view console availability and browse games by providing a fast, simple, and transparent platform.

## Users
### Primary Customers
- **Online customers:** People looking to visit the store who want to check availability and games beforehand.

### User Personas
**Store Customer** (15-40 years old)
- **Role:** Gamer / Customer
- **Context:** Wants to play video games at a physical store but wants to avoid a wasted trip.
- **Pain Points:** Unsure if consoles are available, doesn't want to call, needs clear pricing info.
- **Goals:** Quickly check if their preferred console is free, see what games are installed.

**Store Employee** (18-35 years old)
- **Role:** Admin / Staff
- **Context:** Manages day-to-day operations and customer inquiries.
- **Pain Points:** Frequent phone calls about availability, manual coordination of games per console.
- **Goals:** Easily update console status, manage game catalog, reduce repetitive phone inquiries.

## The Problem
### Operational Inefficiency
Customers frequently call the store to ask about console availability and pricing. This interrupts staff and can be frustrating for customers on hold.
**Our Solution:** A real-time (manually updated) status dashboard.

### Information Gap
Customers don't know which games are installed on which consoles before arriving.
**Our Solution:** A detailed game catalog linked to specific consoles.

## Differentiators
### High Performance
Unlike heavy, ad-laden retail sites, Score Game is optimized for speed (<2s) and works well on slow mobile internet.

### Information-Only Simplicity
Unlike booking platforms that require complex registrations, Score Game provides the essential info without friction.

## Key Features
### Core Features
- **Console Availability Dashboard:** Live status (Available/Reserved) for each console.
- **Pricing Breakdown:** Permanent visibility of session pricing tiers (10min, 30min, 1hr).
- **Game Catalog:** Browseable list of all available games with images and descriptions.
- **Console Details:** Full breakdown of installed games per console.

### Collaboration Features
- **Admin Dashboard:** Simple interface for status updates and catalog management for store employees.
- **Google Login Auth:** Secure and easy access for store staff without public registration.
- **Manual Status Updates:** Instant updates to the public site from the dashboard.
