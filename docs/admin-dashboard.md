# Admin Dashboard

*Access restricted to approved employees only. Google login required.*

---

## Console Management

- **Add new console**  
- **Edit existing console** (name, status, installed games)  
- **Delete console**  
- **Change console status** (Available / Reserved)  
- **Update installed games list** per console

---

## Game Management

- **Add new game** (name, image, description)  
- **Edit game** (name, image, description)  
- **Delete game**

---

## Pricing Text Management

- **Edit the pricing explanation** shown at the top of the Consoles page.

---

## Notes

- All changes are **immediate** (manual update).  
- No customer‑facing actions (no booking, no payments).  
- Multiple employees have the same permissions.  
- New admin accounts must be approved directly in the database.
