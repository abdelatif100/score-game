# Product Mission

## Pitch

Video Game Store Information Website is an information-only MVP that helps local customers check console availability and browse the game catalog by providing real-time operational transparency and clear pricing.

## Users

### Primary Customers

- **Online Customers:** Local gamers who want to check if a console is free or if a specific game is available before visiting the store.

### User Personas

**The Pre-Visit Gamer** (18-35 years old)
- **Role:** Customer
- **Context:** Planning a visit to the physical store to play.
- **Pain Points:** Wasting time traveling to a full store; uncertainty about available games.
- **Goals:** Quickly verify if PS5 #1 is available and see the current pricing.

## The Problem

### Operational Friction
Customers frequently call the store to ask about console availability and game lists, which interrupts employees and frustrates customers.

**Our Solution:** A lightweight, fast website that displays real-time status and a full game catalog.

### Pricing Ambiguity
Customers are often unsure of the pricing tiers (10min vs 1hr) before they arrive.

**Our Solution:** A permanently visible pricing explanation section on the consoles page.

## Differentiators

### High Performance Transparency
Unlike generic business listings, we provide specific, manually updated status (Available/Reserved) for every individual console in the shop.

### Game-to-Console Mapping
We provide a curated preview of installed games for each console, allowing users to make informed decisions before they walk in.

## Key Features

### Core Features
- **Console Availability Display:** Real-time (manually updated) status of all in-store consoles.
- **Pricing Explanation Section:** Clear breakdown of 10/30/60 minute tiers.
- **Game Catalog Grid:** Full browseable list of all games available in the store.
- **Console Details Page:** Complete list of installed games and specific pricing for a selected console.

### Admin Features
- **Manual Status Management:** Simple toggle for "Available" vs "Reserved" for employees.
- **Catalog Management:** Add/Edit/Delete consoles and games via a protected dashboard.
- **Google Login Auth:** Secure, restricted access for store employees only.
