# Score Game

A fast, simple, and transparent video game store information website that helps customers view console availability and browse games before visiting the physical store.

## 🎮 About

**Score Game** is an information-only platform designed to reduce phone inquiries and improve the pre-visit experience for gamers. Unlike complex booking systems, we focus on delivering essential console availability and game catalog information with high performance (<2 seconds load time on mobile).

### The Problem We Solve

- **Operational Inefficiency:** Customers frequently call the store to ask about console availability and pricing
- **Information Gap:** Customers don't know which games are installed on which consoles before arriving

### Our Solution

- Real-time (manually updated) status dashboard showing console availability
- Detailed game catalog linked to specific consoles
- Fast, mobile-optimized platform designed for slow internet connections

---

## 👥 Users

### Store Customers (15-40 years old)
- Want to check if their preferred console is available before visiting
- Need to know what games are installed
- Want clear pricing information without calling

### Store Employees (18-35 years old)
- Manage day-to-day operations and customer inquiries
- Reduce repetitive phone calls about availability
- Easily update console status and manage game catalog

---

## ✨ Key Features

### For Customers
- **Console Availability Dashboard:** Real-time status (Available/Reserved) for each console
- **Pricing Breakdown:** Clear session pricing tiers (10 min, 30 min, 1 hour)
- **Game Catalog:** Browse all available games with images and descriptions
- **Console Details:** View full breakdown of installed games per console

### For Store Staff
- **Admin Dashboard:** Simple interface for status updates and catalog management
- **Google Login Auth:** Secure, easy access without public registration
- **Manual Status Updates:** Instant updates to the public site

---

## 🛠️ Technical Stack

- **Framework:** Next.js 16 (App Router) with React
- **Database:** PostgreSQL
- **Styling:** Tailwind CSS
- **UI Components:** shadcn/ui
- **Icons:** Lucide
- **Fonts:** Google Fonts
- **Frontend Hosting:** Vercel
- **Backend Hosting:** Self-hosted
- **Database:** Self-hosted
- **Deployment:** Vercel CI/CD

## 🗺️ Product Roadmap

### Phase 1: MVP Core
**Goal:** Establish the public website and admin dashboard with essential functionality

- [ ] Public Homepage - Simple landing page with clear navigation
- [ ] Console Availability Dashboard - List showing real-time status and pricing
- [ ] Game Catalog - Browseable grid view of all available games
- [ ] Admin Dashboard - Interface for status updates and game/console management
- [ ] Google Login Auth - Secure admin-only access for staff

### Phase 2: Performance & Polish
**Goal:** Ensure the site is extremely fast and mobile-friendly

- [ ] Image Optimization - Automated compression for game covers and icons
- [ ] SEO & Metadata - Basic search engine optimization for store visibility
- [ ] UI/UX Polishing - Refining the interface with shadcn/ui components

### Phase 3: Future Scale
**Goal:** Add advanced convenience features based on user demand

- [ ] Online Reservation System - Allow customers to book time slots
- [ ] Real-time Notifications - Alert customers when a console becomes free
